# Pharma-Sales-Analytics-PowerBI-SQL
End-to-end Pharma Sales Analytics project using SQL Server and Power BI

## Project Overview
This project demonstrates an end-to-end Pharma Sales Analytics solution
built using SQL Server and Power BI. The goal is to analyze sales trends,
track category-wise performance, and deliver actionable business insights
for pharma commercial decision-making.

---

## Tools & Technologies
- SQL Server (Data storage, transformations)
- Power BI (Data modeling & visualization)
- DAX (Measures & KPIs)
- GitHub (Project version control)

---

## Data Description
The dataset contains monthly pharma sales across multiple therapeutic
categories such as:
- Analgesics
- Anti-inflammatory
- Antihistamines
- Respiratory
- Hypnotics

---

## Key SQL Work
- Created fact and date tables
- Built a star schema data model
- Performed data cleaning and transformations
- Created views for Power BI consumption

---

## Power BI Dashboard Highlights
- Total Sales KPI
- Year-wise sales trend analysis
- Category-wise sales comparison
- Interactive date slicer for dynamic analysis
